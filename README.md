# ewm-mfs-map
A reference model acting as a Map to guide users, consultants and developers through the EWM MFS world.
For further details check the video (https://youtu.be/64m0KrT3IcQ) or blog-post (https://wmexperts.online/blog-posts/mfs/mfs-map.html). 

## 🗺️ How to Contribute to the #MFSMAP
This is a community project! If you have ideas to improve the logic or add components, please contribute.

### 🟢 Method A: The Simple Way (No Setup)
1. Fork this repository to your own GitHub account.
2. Download the `WMexperts-EWM-MFS-MAP.svg` file to your PC.
3. Open it in [draw.io (web version)](https://app.diagrams.net) via File > Open From > Device.
4. Make your changes and Save the file back to your PC.
5. Upload the file to your GitHub fork and create a Pull Request.

### 🔵 Method B: The Direct Way (Cloud Edit)
1. Fork this repository.
2. Use the link below (replace `[YOUR_USERNAME]` with your GitHub name):
   `https://app.diagrams.net/#H[YOUR_USERNAME]/ewm-mfs-map/main/WMexperts-EWM-MFS-MAP.svg`
3. Edit directly in the browser and hit Save to commit to your fork.
4. Create a Pull Request.
